Sincronización Microsoft 365
⚙ M365: configurar credenciales
Conecta con SharePoint de Vitafoods. Los datos se sincronizan en tiempo real para todos los usuarios con el enlace. Usa las licencias M365 que ya tienen.
Pasos para configurar (solo una vez):
1. Ir a portal.azure.com → Azure Active Directory → Registros de aplicaciones → Nueva
2. Nombre: AppProyectosVitafoods · Tipo: Cuentas de esta organización
3. Copiar el Tenant ID y Client ID que aparecen
4. En Permisos de API agregar: Sites.ReadWrite.All (Microsoft Graph)
5. Crear la lista en SharePoint con el mismo nombre configurado
6. Pedir a TI que apruebe los permisos en Azure AD

✅ La forma más simple — JSONBin.io

Sin configurar
Cómo configurar en 3 minutos:
1. Ir a jsonbin.io → crear cuenta gratis
2. Ir a Account → Master Key → copiar
3. Pegar aquí y clic en "Publicar datos actuales"
4. Se genera el Bin ID automáticamente → guardarlo
5. Subir esta app a Netlify → el enlace ya funciona en todos los dispositivos
